# Dog-Identifier-2Layer-NN
Two Layer ML model to recognize images of dogs. <br>

### Installing:
Install uv: ```pip install uv``` <br>
Using uv package manager: ``` uv.sync ``` <br>

### Running:
Use ```uv run src/main.py``` <br>
